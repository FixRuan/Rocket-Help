## Como executar a aplicação 

### Instalação
1. Clone o repositório
```bash
git clone https://github.com/FixRuan/Rocket-Help
```
2. Acesse a pasta do projeto
```bash
cd Rocket-Help
```
3. Instale as dependências necessárias 
```bash
yarn
```
4. Execute o projeto
```bash
expo run:android
```
